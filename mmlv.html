Mega Man Maker Level Editor (MMLV Editor)

Personal Project (2020-2021)

Disclaimer

The video was not made by me. I'm showing this as a reference to demonstrate the overview of my program.

MMLV Editor is an external level editor tool for Mega Man Maker, designed for making existing levels to be more aesthetic and to customize the object properties that the Mega Man Maker can't do. Its main features are:

  • Manual tiling, allowing the user to have more level creation possibilities. In Mega Man Maker, however, uses autotiler that sometimes automatically place the tiles the user don't want.
  • Object editing. For example, a 0.111-second delay laser beam which is not possible to change the precision in the actual level builder.
  • Manual disabling or enabling active screens. Useful for making empty rooms without having to put any object on the screen to activate.
  • Loads level file faster than the Mega Man Maker. A 500kb file takes only about half a second in MMLV Editor where Mega Man Maker take more than a minute to successfully load a level.
  • Save/Load level feature. The program also remove unnecessary decimal values when saving a level, resulting in a smaller file size.
