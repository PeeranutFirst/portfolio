Mathgical Rush

Project Type

Graduation Project

Year

2019-2020

Game Engine

Godot

Team Size

3

Platform

Windows, Android

Mathgical Rush is a 2D puzzle time-attack game on Android and Windows platforms based on the concept of 5th grader lessons (covering the 4th to 6th grades).

The game features a timed minigame for each level. In which the player will have to solve math puzzles before the timer runs out. Passing a puzzle will earn points and accumulate to the next. Each puzzle is randomized and will not be the same every time. The more levels the player pass, the difficulty increases until the game is over.

The game keeps player's record and scores for each plays so that players can improve their calculations skill, better concentration, and better responsiveness. Mathgical Rush can also be used for children having class activities together in the classroom to create a good atmosphere before starting a math lesson and to apply the old knowledge with the new knowledge accordingly.

SceneCtrler.gd
# SceneCtrler (Singleton)
#
# Responsible for managing the queue of the scenes and controls where the
# scene will go to.

class_name SceneCtrler extends Node


var queue : Array setget , get_queue


func reload(): _to_scene()


func go_to(scene : String): _to_scene(scene)
func go_to_splash(): _to_scene("res://src/Scenes/Splash.tscn")
func go_to_main_menu(): _to_scene("res://src/Scenes/MainMenu.tscn")
func go_to_puzzle(): _to_scene("res://src/Scenes/Puzzle.tscn") # An initial scene that manipulates the puzzles order and go to one of them
func go_to_game_over(): _to_scene("res://src/Scenes/GameOver.tscn")


# Add a scene path to the queue. When successfully added, a function
# `next_scene()` can be called.
func add_to_queue(scene_path : String):
    queue.append(scene_path):


func clear_queue():
    queue.clear():


# Go to next scene by pulling a queue from a variable `queue`.
# This method only works if there is at least one element in `queue`.
# If `queue` is empty, an error is thrown.
func next_scene():
    assert(not queue.empty())
    
    go_to(queue.front())
    queue.pop_front()


func has_next() -> bool:
    return not queue.empty()


func _to_scene(scene_path : String = ""):
    get_tree().paused = true
    FadeScreen.start_fading_animation()
    yield(FadeScreen, "enclosed")
    get_tree().change_scene(scene_path)
