Galax Hero (Demo)

Personal Project (2020)

Galax Hero is a work-in-progress RPG Shoot-em-up (shmup) hybrid for the Android platform made by me using Godot Engine. This game is inspired by Shoot-em-up games like Star Force, 1942, and Gun Nac. In the game, the player pilots a spaceship while shooting various enemies. The game features Superpower where every successful hit made by the player’s projectile increases superpower gauge, allowing the ship to unleash the powerful ability when it’s full. In addition, every time an enemy is destroyed, it creates multiple debris that spread and damages nearby enemies, allowing the creation of very devastating combos.

Since many games, sometimes, are easy or difficult to implement which depends on the genre and the number of features in the game. Galax Hero, in my opinion, is the easiest game to code and implement due to the less complex the game currently has. Although the game is very easy to make, writing a clean code that allows the game to be easily expanded can still be difficult.

In the current state of the game, all of the current code can still be optimized and refactored since there are not many features in the game yet. This is a good thing so that I can always go back to practice and improve the current code in the ways the game can be easily expanded.
